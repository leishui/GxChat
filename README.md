# GxChat
## 这是一款界面十分简约的聊天机器人软件，有记住账号密码、储存聊天记录和头像及背景信息的功能，由于是从图灵机器人网获取的免费的机器人，每天好像最多只能交互100句。
### 在联网的前提下只需要在登录界面输入的账号、密码都为12345即可登录。
![登录界面1](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001238.jpg)
![登录界面2](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001301.jpg)
登录成功后即可看到一个如图正在旋转的滑稽发来的一条消息
![登录成功](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001307.jpg)
### 在有网络的环境下即可进行聊天
![聊天1](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001350.jpg)
![聊天2](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001400.jpg)
### 点击右上角的按钮就会弹出三个选项（暂时为三个），如图，分别为“清除消息列表”、“设置头像”、“设置背景”
![三选项](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001409.jpg)
#### 首先介绍设置头像，点击选项后可以让你从相册选一张图片作为你的头像（静态图动态图都可以，但目前只支持设置方形头像，且暂无裁剪功能），效果如图
![骚](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-0015491.jpg)
![设置头像](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001534.jpg)
### 再就是设置背景，点击选项后可以更改聊天背景，同为从相册选一张图片，但暂不支持动态图，效果如图
![设置背景](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001653.jpg)
### 最后就是清除消息列表了，就是把原本储存下来的聊天记录删掉（所有储存下来的记录）
![清除消息列表](https://github.com/leishui/GxChat/blob/master/image/Screenshot_20190302-001803.jpg)
### 总而言之，目前能力有限，也在努力尝试添加一些新功能，希望通过以后的逐步完善，把它做成真正实用的作品。
